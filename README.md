# P4 無の氾濫練習

FF14「絶妖星乱舞」P4 グランドクロス内の「無の氾濫」を練習するためのページです。

3回目グランドクロスのビーム判断から、無の氾濫で実際に入るビーム位置までを確認できます。

## できること

- 3回目グランドクロスのギミック理解
- 無の氾濫の位置確認
- 無の氾濫の実践練習
- 「無の氾濫単体」と「GC3＋無の氾濫」の切り替え
- 自分ロールの切り替え
- パーティーリスト上の真偽表示、詳細表示の切り替え
- 正解表示、前の問題、別の問題、成績リセット

## モード

### ギミック理解

3回目グランドクロスのデバフと真偽を見て、入るビームを選択肢で判断します。

- 死の超越
- アラガンフィールド
- 傷の色
- エクスデス真偽
- ビーム真偽

### 位置確認

マップをクリックして、無の氾濫で入る位置を確認します。

範囲は2種類あります。

| 範囲 | 内容 |
| --- | --- |
| 無の氾濫単体 | 無の氾濫の位置だけを確認 |
| GC3＋無の氾濫 | 直前の3回目GC読取から、無の氾濫の位置までを2枚で確認 |

### 実践練習

3カウント後にタイマーが始まり、制限時間内に回答します。

デフォルトは「無の氾濫単体」です。

| 範囲 | 枚数 | 内容 | 秒数 |
| --- | --- | --- | --- |
| 無の氾濫単体 | 1 | 無の氾濫 | 5秒 |
| GC3＋無の氾濫 | 1 | 3回目GC読取 | 4秒 |
| GC3＋無の氾濫 | 2 | 無の氾濫 | 11秒 |

「一問ごと」のチェックを外した状態がデフォルトです。

## 無の氾濫の基本

- 死の超越は、デバフと同色ビーム
- アラガンフィールドは、デバフと逆色ビーム
- ビームが偽なら、最終的に入るビーム色を反転
- THはエクスデス側
- DPSはエクスデスと反対側

## 操作

- 自分ロール: 自分の担当ロールを変更
- 範囲: 無の氾濫単体 / GC3＋無の氾濫を切り替え
- 前の問題: ひとつ前の問題へ戻る
- 別の問題: 新しい問題を出す
- 正解を表示: 正解位置を表示
- 一問ごと: 実践練習の結果表示後に自動で次へ進むかを切り替え
- 成績リセット: 現在のモード・範囲の成績をリセット
